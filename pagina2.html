Vamos aprender HTML!

Oque e HTML?

(Hyper Text Markup Language) é a linguagem padrão usada para criar e estruturar páginas web. É o "esqueleto" de um site, definindo a disposição de elementos como textos, imagens, links e outros conteúdos. Não é uma linguagem de programação, mas sim uma linguagem de marcação que organiza e formata o conteúdo para que os navegadores possam interpretá-lo corretamente.

Sugestão de Tags essenciais

Para organizar as tags, vamos separá-las em categorias e usar uma estrutura de lista para facilitar a visualização e o aprendizado.

Estrutura Básica do Documento

<!DOCTYPE html>:

Função: Informa ao navegador que o documento está escrito em HTML5, a versão mais recente da linguagem.
Observação: É essencial colocar esta declaração no início de todo documento HTML, pois ela ajuda o navegador a renderizar a página corretamente.
Exemplo:<!DOCTYPE html>

<head>:

Função: Conhecida como cabeça, serve para inserir informações sobre o documento, como título, dados para mecanismos de busca e links para arquivos CSS e JavaScript.
Fechamento: </head>
Exemplo:
<head>
<meta charset="utf-8">
<title>Meu Título</title>
<link rel="stylesheet" href="style.css">
</head>

<title>:

Função: Define o título de uma página da web, que aparece na aba do navegador e nos resultados de pesquisa, sendo um fator crucial para SEO (uso de elementos e atributos HTML para otimizar páginas web para mecanismos de busca, melhorando sua visibilidade e classificação nos resultados de pesquisa) e a experiência do usuário.
Fechamento: </title>
Exemplo:<title>Minha Primeira Página</title>

<body>:

Função: Contém todo o conteúdo visível da página web, como textos, imagens, links, vídeos, etc. É onde você constrói o corpo do seu site.
Fechamento: </body>
Exemplo:
<body>
<h1>Bem-vindo!</h1>
<p>Este é o conteúdo da minha página.</p>
</body>
Tags de Texto e Formatação

<h1>a<h6>:

Função: Usadas para definir títulos e subtítulos em uma página ou seção. <h1> é o título principal (o mais importante) e <h6> é o menos importante, seguindo uma hierarquia.
Fechamento: </h1> a </h6>
Exemplo:
<h1>Título Principal</h1>
<h2>Subtítulo Importante</h2>
<h6>Subtítulo Menos Importante</h6>

<p>:

Função: Conhecido como parágrafo, é um elemento de bloco, o que significa que ocupa toda a largura disponível e cria uma quebra de linha antes e depois do conteúdo.
Fechamento: </p>
Exemplo:
<p>Este é um parágrafo de texto em HTML.</p>
<p>Aqui está outro parágrafo.</p>

<br>:

Função: Usada para inserir uma quebra de linha em um texto. É uma tag vazia, ou seja, não precisa de uma tag de fechamento.
Exemplo: Este é um texto.<br>
Ele terá uma quebra de linha aqui.

<em>:

Função: Usada para marcar texto que tem ênfase, normalmente exibido em itálico pelo navegador.
Fechamento:
Exemplo: Você deve aprender <em>HTML</em>.

<strong>:

Função: Usada para definir texto com importância forte ou relevância. É interpretada pelos navegadorescomo um texto que deve ter destaque visual, geralmente em negrito, mas seu propósito principal ésemântico, indicando que o conteúdo dentro dela é mais importante do que o texto ao redor.
Fechamento: </strong>
Exemplo:
<;strong>Atenção:</strong> Salve seu trabalho frequentemente.
Tags de Mídia e Interatividade

<img>:

Função: Usada para colocar uma única imagem em uma página da web. É um elemento vazio (não possui conteúdo de texto ou tag de fechamento).
Atributo src: Especifica o caminho ou URL da imagem.
Atributo alt: Fornece um texto alternativo para a imagem, usado em caso de falha no carregamento da imagem ou por leitores de tela para pessoas com deficiência visual.
Atributos de dimensão (width, height): Podem ser usados para especificar a largura e a altura de um elemento, geralmente definidos com CSS.
Exemplo: <img src="minha-imagem.jpg" alt="Descrição da imagem" width="300" height="200">

<a>:

Função: Conhecida como tag de âncora (Anchor), é responsável por vincular os demais documentos que o nosso HTML possui, links internos e externos.
Atributo href: Especifica o destino do link, ou seja, para onde o usuário será direcionado quando clicar no link. O href pode referenciar uma URL externa, um arquivo, um endereço de e-mail, ou até mesmo uma seção dentro da mesma página.
Fechamento: </a>
Exemplo:
<a href="https://www.google.com">Visite o Google</a>
<a href="contato.html">Entre em contato</a>

<button>:

Função: Usada para criar um botão clicável em páginas web, geralmente para acionar ações como enviar formulários ou executar funções JavaScript. Ela pode ser estilizada com CSS e, ao contrário da tag <a>, é mais adequada para interações dentro da própria página, enquanto <a> é usada para links externos ou navegação entre páginas.
Fechamento: </button>
Exemplo: <button></button>
Tags de Estilo

<style>:

Função: Usada para definir informações de estilo, como regras CSS (Cascading Style Sheets), diretamente dentro de um documento HTML. Essa tag permite controlar a aparência visual dos elementos da página, como cores, fontes e layout, sem a necessidade de arquivos CSS externos.
Fechamento: </style>
Exemplo:
<head>
<style>
body {
font-family: Arial, sans-serif;
background-color: lightblue;
}
h1 {
color: navy;
}
</style>
</head>
