Tags HTML

<!DOCTYPE html>:

é usada para informar ao navegador que o documento está escrito em HTML5, a versão mais recente da linguagem. É essencial colocar esta declaração no início de todo documento HTML, pois ela ajuda o navegador a renderizar a página corretamente.

<head>:

conhecida como cabeça, serve para inserir informações sobre o documento, como título, dados para mecanismos de busca e links para arquivos CSS e JavaScript.

</head>:

Tag de fechamento da cabeça:

<p>:

conhecido como parágrafo, é um elemento de bloco, o que significa que ocupa toda a largura disponível e cria uma quebra de linha antes e depois do conteúdo.

</p>

Tag de fechamento para o parágrafo:

<br>:

é usada para inserir uma quebra de linha em um texto. Ela é uma tag vazia, ou seja, não precisa de uma tag de fechamento.

<h1>:

é usada para definir o título principal de uma página ou seção.

</h1>:

Tag de fechamento para o título principal:

<h6>:

é uma tag HTML utilizada para definir o sexto nível de cabeçalho em uma página web.Ela faz parte da hierarquia de tags de cabeçalho, que vão de <h1> (o mais importante) até <h6> (o menos importante).

</6>:

Tag de fechamento para o sexto nível de cabeçalho:

<em>:

é usada para marcar texto que tem ênfase , normalmente exibido em itálico pelo navegador.

</em>:

Tag de fechamento para ênfase:

<strong>:

é usada para definir texto com importância forte ou relevância. Ela é interpretada pelos navegadores como um texto que deve ter destaque visual, geralmente em negrito, mas seu propósito principal é semântico, indicando que o conteúdo dentro dela é mais importante do que o texto ao redor.

</em>:

Tag de fechamento para ênfase:

<button>:

é usada para criar um botão clicável em páginas web, geralmente para acionar ações como enviar formulários ou executar funções JavaScript. Ela pode ser estilizada com CSS e, ao contrário da tag <a>, é mais adequada para interações dentro da própria página, enquanto <a> é usada para links externos ou navegação entre páginas.

</strong>:

Tag de fechamento para botão:

<style>:

é usada para definir informações de estilo, como regras CSS (Cascading Style Sheets), diretamente dentro de um documento HTML. Essa tag permite controlar a aparência visual dos elementos da página, como cores, fontes e layout, sem a necessidade de arquivos CSS externos.

<style>:

Tag de fechamento para estilo:

<a href="link">:

A tag <a> em HTML é usada para criar links, e o atributo href especifica o destino do link, ou seja, para onde o usuário será direcionado quando clicar no link. O href pode referenciar uma URL externa, um arquivo, um endereço de e-mail, ou até mesmo uma seção dentro da mesma página.

<title>:

define o título de uma página da web, que aparece na aba do navegador e nos resultados de pesquisa, sendo um fator crucial para SEO e a experiência do usuário.
