Oque e css?

CSS (Cascading Style Sheets) é uma >linguagem de estilo usada para descrever a apresentação de um documento escrito em HTML ou XML (como páginas web). Simplificando, enquanto o HTML estrutura o conteúdo de um site (como textos, imagens, etc.), o CSS controla como esse conteúdo é exibido, definindo aspectos visuais como cores, fontes, layouts e espaçamentos.

Sugestão de Conceitos e Propriedades CSS Essenciais

Para facilitar o aprendizado, vamos dividir os conceitos e propriedades CSS em categorias.

Como Conectar CSS ao HTML?

Existem três maneiras principais de aplicar CSS a um documento HTML:

1.CSS Externo (Recomendado)

Função: É a forma mais comum e recomendada. Você cria um arquivo .css separado e o vincula ao seu HTML usando a tag <link> dentro da seção <head>.
Vantagem: Facilita a manutenção do código, pois você pode alterar o estilo de várias páginas HTML de uma só vez.
Exemplo no HTML (<head>):<link rel="stylesheet" href="style.css">
Exemplo no arquivo style.css: <p>
body {
font-family: Arial, sans-serif;
background-color: lightblue;
}
h1 {
color: navy;
}
</p>

2.CSS Interno

Função: Você define as regras CSS diretamente no documento HTML, dentro da tag <style> na seção <head>.
Vantagem: Útil para estilos específicos de uma única página.
Exemplo no HTML (<head>):
<head>
<style>
p {
color: green;
font-size: 16px;
}
</style>
</head>

3.CSS Inline

Função: As regras CSS são aplicadas diretamente a um elemento HTML específico usando o atributo style.
Vantagemv: Útil para estilizar um único elemento sem afetar outros.
Desvantagem: Não é recomendado para estilos complexos ou reutilizáveis , pois mistura HTML e CSS.
Exemplo no HTML: <p style="color: blue; text-align: center;">Este parágrafo está azul e centralizado.</p>

Seletores CSS

Seletores são usados para "selecionar" os elementos HTML que você quer estilizar.
Seletor de Elemento (Tipo)
Função: Seleciona todos os elementos de um determinado tipo.
Exemplo:
p { /* Seleciona todos os parágrafos */
color: red;
}

Seletor de Classe

Função: Seleciona elementos que possuem um atributo <class> específico. Um mesmo nome de classe pode ser usado em vários elementos.
No HTML: <p class="destaque">Texto em destaque</p>
No CSS:
.destaque { /* O ponto (.) indica que é uma classe */
background-color: yellow;
}

Seletor de ID

Função: Seleciona um único elemento com um atributo id específico. Cada id deve ser único por página.
No HTML: <h1 id="titulo-principal">Meu Título</h1>
No CSS:
#titulo-principal { /* A hashtag (#) indica que é um ID */
font-size: 40px;
text-align: center;
}
Propriedades CSS Comuns

Aqui estão algumas propriedades CSS fundamentais que você usará frequentemente:

Propriedades de Texto e Fonte

<color>

Função: Define a cor do texto.
Exemplo: color: blue; ou color: #FF0000; ou color: rgb(255, 0, 0);

<font-family>

Função: Define o tipo de fonte a ser usado.
Exemplo: font-family: Arial, sans-serif;

<font-size>

Função: Define o tamanho da fonte.
Exemplo: font-size: 16px; ou font-size: 1.2em;

<font-weight>

Função: Define a espessura da fonte (negrito ou normal).
Exemplo: font-weight: bold; ou font-weight: 700;

<text-align>

Função: Alinha o texto dentro de um elemento.
Valores: left, right, center, justify.
Exemplo: text-align: center;

<text-decoration>

Função: Adiciona ou remove decorações de texto, como sublinhado.
Valores: none, underline, overline, line-through.
Exemplo: text-decoration: none; (muito comum para remover sublinhado de links)
Propriedades de Caixa (Box Model)

O Box Model é um conceito fundamental no CSS que descreve como os elementos HTML são renderizados como caixas retangulares.

width e height

Função: Definem a largura e a altura de um elemento.
Exemplo: width: 200px; height: 150px;

padding

Função: Define o espaçamento interno entre o conteúdo de um elemento e sua borda.
Exemplo: padding: 10px; (para todos os lados) ou padding: 10px 20px; (topo/fundo, direita/esquerda).

margin

Função: Define o espaçamento externo ao redor de um elemento, criando espaço entre ele e outros elementos.
Exemplo: margin: 20px; (para todos os lados) ou margin: 0 auto; (para centralizar um bloco).

border

Função: Define a borda de um elemento.
Exemplo: border: 1px solid black; (espessura, estilo, cor).
Propriedades de Fundo e Cor

background-color

Função: Define a cor de fundo de um elemento.
Exemplo: background-color: #F0F8FF;

background-image

Função: Define uma imagem como fundo de um elemento.
Exemplo: background-image: url('imagem-de-fundo.png');
Propriedades de Layout (Básico)

display

Função: Define como um elemento é exibido.
block:Ocupa toda a largura disponível e força uma nova linha (ex: <div>, <p> ).
inline:Ocupa apenas o espaço necessário e não força uma nova linha (ex: <span>,<a> ).
inline-block:Combina características de inline e block (permite definir largura e altura, mas se mantém na mesma linha).
none: Oculta o elemento completamente.
Exemplo:display: block; ou display: inline-block;
